# KIC21

## 📌 개요  
`KIC21`는 HTML, SCSS, JavaScript 기반의 웹 프로젝트로, 웹페이지 콘텐츠 관리 및 사이트 구성에 활용할 수 있는 구조를 제공합니다.  
정적 리소스, 컴포넌트, 템플릿 등을 모듈 단위로 관리할 수 있게 설계되어 있습니다.

---

## 🗂️ 폴더 구조  
- assets/ ─ CSS, 이미지, 폰트 등 정적 리소스
- html/ ─ 페이지 HTML 파일들
- js/ ─ JavaScript 기능 스크립트
- scss/ ─ SCSS 스타일 원본 파일


---

## 🛠️ 기술 스택  
- HTML5  
- SCSS → CSS (모듈형 구조)  
- JavaScript (ES6+)  
- GitHub (버전 관리)

---

## 🚀 설치 및 실행 방법  
1. 레포지토리를 클론합니다.  
    git clone https://github.com/okkim90/kic21.git
2. 프로젝트 폴더로 이동합니다. 
    cd cms_m
3. SCSS를 CSS로 컴파일합니다. (Live Sass Compiler 또는 npm 스크립트 사용)
    npm install
    npm run build
4. 브라우저에서 html/index.html을 열어 확인합니다.

---

## 🧩 사용 예시

- HTML 템플릿 기반 CMS 페이지 구성
- 프론트엔드 퍼블리싱 구조 참고용
- SCSS 모듈화 및 컴포넌트 관리 예시
